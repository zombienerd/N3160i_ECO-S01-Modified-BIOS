# N3160i_ECO-S01-Modified-BIOS
A modified "Stock" bios for the MSI N3160i_ECO-S01 from the Barracuda 290a storage appliance.
Created by splicing the N3160 microcode into the MSI N3150 ECO stock Bios latest revision (E7890IMS.130) 
Must be flashed manually from DOS or AMI Bios EFI shim, and given all the overrides.
